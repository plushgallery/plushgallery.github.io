What is Bootstrap?

Bootstrap is an open source, free front-end web development tool for developing websites. It uses a variety of customizable HTML and CSS templates, as well as workable Javascript extensions. Some of the Bootstrap templates include Starter, Bootstrap Theme, Grid, Jumbotron, and Narrow Jumbotron. Source code for each of these frameworks, as well as nav bars and a variety of components is available at the download page. Current versions of Bootstrap also support responsive web design.
