What is Bootstrap?

Bootstrap is an open source, free front-end web development tool for developing websites. It uses a variety of customizable HTML and CSS templates, as well as workable Javascript extensions. Source code for each of these frameworks, as well as nav bars and a variety of components is available at the download page. Current versions of Bootstrap also support responsive web design.

Text Editor / IDE

Robust text editors such as Sublime Text 2 can be used as integrated development environments for writing and editing HTML, CSS, JavaScript and more. Useful features include split screen functionality, customizable browser previews, coordinated color schemes to easily read your code, and keyboard shortcuts.

Assignment

In this assignment, we will work with the Bootstrap Jumbotron template, adding a background image, modifying the email response fields using a jQuery script, customizing the buttons and testing the page for responsive web design. We will also test the page through the W3C validator service to check for how it conforms to standards.
